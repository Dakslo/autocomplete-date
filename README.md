# autocomplete-date package

> autocomplete date

<img src="https://raw.githubusercontent.com/akameco/autocomplete-date/master/screenshot.gif"/>

## Install

```
$ apm install autocomplete-date
```

## Usage
Provide snippet

### date (dd)
`YYYY-MM-DD`

### time (dt)
`hh:mm:ss`

### now
`YYYY-MM-DD hh:mm:ss`
